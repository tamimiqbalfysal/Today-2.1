# **App Name**: Simple Sign Up

## Core Features:

- User Registration: Allow new users to create accounts by providing name, username, email, country, date of birth, and password.
- Input Validation: Validate user input on the client-side to ensure all fields are properly filled before submission.
- Account Creation: Securely store user information (after validation) in Firestore upon successful account creation.
- Login Redirection: Redirect newly registered users to the login page after successful registration.

## Style Guidelines:

- Primary color: Dark navy blue (#1A237E) to convey trustworthiness and seriousness.
- Background color: Light gray (#F5F5F5) to create a clean and simple backdrop.
- Accent color: Indigo (#3F51B5) for button highlights and interactive elements.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif, is used to create a modern yet approachable feel, consistent with the image provided
- Use a centered layout for the sign-up form to maintain focus.
- Employ consistent spacing and padding for a clean, orderly appearance.
